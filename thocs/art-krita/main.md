Commands/Shortcuts
====================================================================
- Control + Pencil = Zoom
- Shift + Pencil = Tilt
- Ctrl + A : select all
- Ctrl + H : show selection
- PgUP     : next layer
- PgDown   : previous layer
- [        : Increase brush
- ]        : Decrease brush
- Shift + B: Block layer
- Shift + V: View layer
- B        : brush
- T        : Move layer
- Shift + P: pointer (no tool)
- Ctrl + R : rectangular selection tool
- Insert   : new layer
- Ctrl + E : Merge layers
